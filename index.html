PQ/QBS/TP/SOQ 독소조항 탐지기 · 사전확장판

형식 제한과 감점 문구는 물론, 실적·자격 과다 제한·과다 배상책임·지식재산권 부당이관 문구까지 함께 찾아내고, 사용자 키워드와 독소조항 누적 학습 문구를 반영해 위험도를 정교하게 산출하는 버전입니다. PDF / HWPX / TXT / MD / HTML 업로드를 지원하며, Gemini API 키를 설정하면 분석 직후 AI가 자동으로 재검토하여 실제 독소조항으로 확인된 문구만 골라서 보여줍니다.
점수 높은 순 정렬 스마트 문단 추출 실적·자격 과다제한 탐지 AI 자동 필터링(선택) 전 직원 오탐지 화이트리스트 통계 대시보드(최근 6개월) 엑셀 체크리스트 내보내기

실무 메모

  • 형식 제한 + 감점 문구가 결합되면 위험도 가중
  • 실적·자격 과다제한, 과다 배상책임, 지식재산권 부당이관도 별도 카테고리로 탐지
  • 학습 문구와 유사한 줄은 추가 가점, 표준 기준서 대비 추가 문구는 가중치 10 부여
  • 사용자 키워드·학습 문구는 전 직원 공유 DB에 저장되어 재접속 후에도 유지
  • 5명 이상이 같은 문구를 오탐지로 신고하면 관리자 승인 후 전사 화이트리스트로 자동 정화

🤖 AI 자동 필터링 (Gemini, 전 직원 공통 적용)

[독소조항 분석] 버튼을 누르면 사전·정규식으로 탐지된 후보 문구들을 사내 AI 프록시(Cloudflare Worker)를 거쳐 Google Gemini가 다시 검토하여, 실제로 독소조항이라고 판단한 문구만 최종 목록에 남깁니다. Gemini API 키는 이 파일이나 브라우저 어디에도 저장되지 않고 서버(Worker) 안에만 있으며, 각 직원이 개별로 키를 설정할 필요 없이 관리자가 한 번 배포해두면 전 직원에게 자동 적용됩니다. 탐지된 후보 문구(및 앞뒤 한 줄)만 전송되고 문서 전체는 전송되지 않습니다.
